# RunningPlaner
Простий план тренувань для бігу, можна бачити і вести свій прогрес . Має два плани тренувань, за цільвою датою і адаптивний, коли враховуються реальні показники бігуна. 
Можна виставити цільову дату, якийсь проміжний івент, наприклад в кінці марафон, а по середині напівмарафон, чи щось на кшталт цього.
